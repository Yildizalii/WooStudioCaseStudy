## Project Tech Stack
* Xcode: Version 13.4.1
* Language: Swift 5.6.1
* Design Pattern: MVVM
* Dependency  Manager: Swift Package Manager
* Dependencies: Alamofire, AlamofireImage
* Powered by [Pokeapi](https://pokeapi.co)

## Structure
```bash
⊢ Network
  ⊢ NetworkService
⊢ Scenes
  ⊢ Launch
  ⊢ List
    ⊢ Model // Entity for API
    ⊢ ABCViewModel.swift
    ⊢ ABCViewController.swift
    ⊢ ABC.storyboard
```
